1) Create .env file inside operationhub folder
2) add following command in .env

MONGODB_URI=mongodb+srv://<<YOUR_MONGODB_URL>>/PAF?retryWrites=true&w=majority&appName=Cluster0

3) Run project and doing the normal registration (create USER)
